🏠 DHL Vetting Tracker

Landing Page + Vetting Flow Pages

Components prontos para integração em seu Next.js

✅ Repositório Ativo

🎯 O que é?

Este repositório contém páginas e componentes completos e reutilizáveis para o DHL Vetting Tracker, construído com:

  • Next.js 16.2+
  • React 19
  • TypeScript
  • Firebase

Pronto para copiar e integrar em qualquer projeto Next.js

📁 Estrutura

  • landing/ - Página inicial com hero, serviços e contato
  • vetting/ - Fluxo de vetting para motoristas
  • components/ - Componentes React reutilizáveis

Todos os componentes usam CSS Modules e Framer Motion

🚀 Quick Start

Para usar este repositório em seu projeto:

git clone https://github.com/j-mazini/dhl-vetting-pages.git

Copie a pasta pages para seu projeto Next.js

Next.js 16.2+

📄 Páginas Disponíveis

🏠 Landing Page

Página inicial com navegação, hero section com animações, abas de serviços, informações da frota e formulário de contato.

🔐 Vetting Flow

Portal do motorista com autenticação Firebase, pré-registro, dashboard de progresso e gerenciamento de documentos.

✨ Componentes

Header, Hero Section, Service Tabs, Fleet Spotlight, Contact Form, Forms de Registro e Histórico de Trabalho.

🎨 Design

Responsivo, animações com Framer Motion, CSS Modules para isolamento de estilos, suporte a TailwindCSS.

📦 Integração

Como usar em seu projeto:

  1. Clone este repositório
  2. Copie a pasta pages/ para seu projeto Next.js
  3. Atualize os imports em seu src/app/
  4. Configure Firebase em src/lib/firebase.ts
  5. Pronto! Use os componentes em suas rotas
